# Running the Project

## 1. Backend:

- Navigate to the `backend/` directory.
- Install the dependencies:

```sh
pip install -r requirements.txt
```

- Run the Flask app:

```sh
python app.py
```

## 2. Frontend:

- Navigate to the `frontend/` directory.
- Install the dependencies:

```sh
npm install
```

- Run the Next app:

```sh
npm run dev
```
